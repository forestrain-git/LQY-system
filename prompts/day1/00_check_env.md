# Day 1 - Prompt 0: 环境检查

**时机**：启动Claude Code后首先执行
**预期耗时**：2分钟
**人工决策**：确认环境OK后继续

---

## 输入Prompt

```text
请先检查当前环境状态，输出诊断报告：

1. 检查Python版本（需要3.10+）
2. 检查Docker是否运行（docker ps）
3. 检查8000/5432/6379端口是否被占用（lsof -i :8000 或 netstat）
4. 检查当前目录结构（ls -la）
5. 检查Git状态（git status）

输出格式：
```
环境诊断报告
==============
Python: 3.11.2 [OK/失败]
Docker: 运行中 [OK/失败]
端口8000: 空闲 [OK/被占用 by xxx]
端口5432: 空闲 [OK/被占用]
端口6379: 空闲 [OK/被占用]
当前目录: /mnt/d/projects/lqy-system [OK]
Git: 已初始化 [OK/未初始化]

结论: [可以开始/需要修复]
修复建议: [如果有问题]
```

如果任何检查失败，请：
- 给出明确的错误信息
- 提供一键修复命令（如果可能）
- 或者给出替代方案（如果无法自动修复）

确认环境OK后，回复"环境检查通过，可以开始Day 1"。
```

## 你的决策

- [ ] 环境OK → 继续Prompt 1
- [ ] 端口被占用 → 决定修改端口或杀掉进程
- [ ] Docker未启动 → 启动Docker后重试
- [ ] Python版本低 → 决定使用conda/virtualenv升级
